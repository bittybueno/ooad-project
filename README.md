# ooad-project

An Object Oriented Cafe System
- sales
- inventory
- rewards membership


Design Patterns:
- Factory
  - creating products (beverages & pastries)

- Template
  - chef & barista

- Command
  - orders

- Decorator
  - pricing for add-ons

- Observer
  - low inventory
  
  ## Tools
  - Written in Java 8 using SDK 11. Developed using VS Code.
